Use Logic :
#
(#) for Title
ex. #Title
#
({}) for Link
ex. {link}
#
([]) for descriptions
ex. [descriptions]
#
(@) to separate block from another blocks
ex.
block  
@  
block  
#
In a block there are 3 types, namely Title, Link, and Description  
In a block Title is only one
In a block Link can have more than one  
In a block Description can have more than one
#
(every line beetwen [inside] will reflect new line in html output)
